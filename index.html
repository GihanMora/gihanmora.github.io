Projects

Dynamic Phylogenetic Tree Construction Using k-mer and Locality Sensitive Hashing (Final Year Research Project)

Developed novel, accurate and efficient method to genetic distance calculation, phylogenetic tree construction and phylogenetic tree updation. Branched to areas such as Bioinformatics, Supervised and Unsupervised Machine Learning.

Stack : Python, DSK tool, TensorFlow, HD5, Django, HTML, CSS, JavaScript.

Analysing Crowdsourcing Requesters’ Wage Setting Behaviors (Internship Research Project)

A crowdsourcing toolkit which has advanced features for requesters to manage tasks and investigate task and worker related behaviours. Also has the capability to predict worker ability, task quality and hourly wage. Branched to areas such as Crowdsourcing, Bayesian Inference, Data Science and Analytics

Stack : Python, AdobeXD, R, Theano, Postgres, BOTO3 SDK, PYMC3, Django, HTML, CSS, JavaScript, AWS services.

REAL-TIME INTELLIGENT SURVEILLANCE FOR CCTV CAMERAS

This system is designed to do intelligent tasks such as detect people, track people with occlusions, monitoring people entrance to regions and analyse people density with time, etc based on the information acquired from the ordinary video feed of a CCTV camera. Also system has the capability to trigger events on business and security aspects, from information acquired.

Stack : Python, OpenCV, Dlib, Caffe Neural Framework, Flask, CSS, JavaScript.

CHILD BICYCLE TRACKING SYSTEM

Smart system that can be used by parents to monitor their children while they are playing with bicycles to ensure the safety. Identify child location, set safe ranges, detect bicycle falling, calling ability are the main functionalities of the system.

Stack : Firebase, Android and Arduino

WEB BASED MANAGEMENT SYSTEM FOR HASINI GARMENTS

A web based system to conveniently manage activities within the garment such as billing payment recording, material transactions recording, salary calculating and recording, invoice reporting, donation and bonus payments, managing worker profiles.

Stack : Python, DSK tool, TensorFlow, HD5, Django, HTML, CSS, JavaScript.

NOTE PICKER- NOTE KEEPER + SMART SCANNER(OCR DETECTION)

Note Picker is simple and smart note keeping android application which supports optical character recognition. Give the convenience to the user to take down notes simply scanning from camera.
